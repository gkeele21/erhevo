## Color Palette

### Primary Palette – Deep Ocean
| Color | Hex | Usage |
|-------|-----|-------|
| Dark Navy | #1A365D | Headings, main text, key UI elements |
| Medium Teal | #136F74 | Secondary text, icons, small highlights |
| Light Aquamarine | #39B8CF | Subtle accents, hover effects |

### Accent Palette – Golden Glow
| Color | Hex | Usage |
|-------|-----|-------|
| Gold | #F0C56F | Buttons, highlights, call-to-action elements |
| Amber | #F4933D | Buttons, hover effects, links |
| Light Ivory | #FFF4DD | Background elements, card highlights |

### Neutral Background
| Color | Hex | Usage |
|-------|-----|-------|
| Soft White | #FAFAFA | Page backgrounds, cards, sections |

### Text Colors
- Primary Text: Dark Navy #1A365D
- Secondary Text: Medium Teal #136F74
- Button / Link Text: White #FFFFFF over accent buttons

## Usage Guidelines
- Use Deep Ocean for main text and primary UI elements.
- Use Golden Glow for accent elements, buttons, and call-to-action highlights.
- Use Soft White for backgrounds to maintain readability and visual comfort.
- Maintain contrast ratios for accessibility, especially on buttons and hover states.
